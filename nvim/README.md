## 🎯 核心快捷键速查

| 快捷键 | 功能 |
|--------|------|
| `<Space>` | Leader 键 |
| `jk` | 退出插入模式 |
| `<leader>w` | 保存文件 |
| `<leader>e` | 文件树 |
| `<leader>f` | 搜索文件 |
| `<leader>F` | 搜索内容 |
| `<leader>t` | 浮动终端 |
| `<leader>ld` | 跳转定义 |
| `<leader>lh` | 显示文档 |
| `<leader>lr` | 重命名 |

详细快捷键请查看 [使用指南与快捷键.md](./使用指南与快捷键.md)
